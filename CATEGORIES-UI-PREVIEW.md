### Tag States

1. **Default (Unselected)**
   - Background: White
   - Border: 2px solid gray-300
   - Text: Gray-700
   - Cursor: Pointer

2. **Hover (Unselected)**
   - Background: Green-50 (teaching) / Blue-50 (learning)
   - Border: 2px solid green-400 / blue-400
   - Text: Gray-700
   - Transform: None

   - Background: Green-600 (teaching) / Blue-600 (learning)
   - Border: None
   - Text: White with checkmark icon
   - Ring: 2px ring-green-400 / ring-blue-400
   - Shadow: Medium shadow
   - Transform: Scale-105 (5% larger)

4. **Hover (Selected)**
   - Same as Selected state
   - Cursor: Pointer (can deselect)
